# Expense Tracker - Statement

**Problem Statement:**  
Students often struggle to track daily spending, making budgeting and saving hard. There’s a need for a simple tool to record and review expenses.

**Scope:**  
A lightweight app for students to log, categorize, and summarize their expenses. Runs on any system with Python—easy to set up and use.

**Target Users:**  
- Students
- Young adults learning to manage money

**High-Level Features:**  
- Add, edit, delete expense entries  
- Categorize expenses  
- View daily/weekly summaries  
- Export expense data  
- Basic input validation
